## Project Overview
This project showcases a **Tableau dashboard** for Moretto Inc, designed to analyze sales performance, customer behavior, and profitability across different regions and product categories. The dashboard allows users to interactively explore key business metrics and make **data-driven decisions**.

## Dataset Description
The dataset contains transactional information for Moretto Inc orders, structured to support sales and profitability analysis.

<img width="1901" height="793" alt="Dataset" src="https://github.com/user-attachments/assets/94b9e3da-d43e-46b0-8955-eec12223cb1d" />

**Columns / Features:**

| Column Name         | Description |
|--------------------|-------------|
| `order_id`          | Unique identifier for each order |
| `customer_id`       | Unique identifier for each customer |
| `customer_name`     | Name of the customer |
| `region`            | Geographical region of the customer (e.g., North, South, East, West) |
| `state`             | State of the customer within the region |
| `order_channel`     | Channel through which the order was placed (e.g., Online, In-store, Distributor) |
| `product_id`        | Unique identifier for each product |
| `product_name`      | Name of the product |
| `product_category`  | Category to which the product belongs (e.g., Electronics, Apparel) |
| `order_date`        | Date when the order was placed |
| `cost_price`        | Cost incurred by the company to produce or acquire the product |
| `unit_price`        | Price per unit at which the product is sold |
| `quantity`          | Number of units purchased in the order |
| `sales_price`       | Total sales price for the order (`unit_price × quantity`) |
| `profit`            | Profit earned from the order (`sales_price – cost_price × quantity`) |

**Dataset Purpose:**
- Track **sales trends** over time by region, state, and channel  
- Identify **high-performing products and categories**  
- Evaluate **customer behavior** and sales patterns  
- Measure **profitability and margins** at order, product, and region levels  

**Format:** CSV or Excel  
**Size:** Thousands of rows (depending on dataset)

## Tableau Dashboard
The dashboard provides insights through interactive visualizations, including:

- **Sales Trend Analysis:** Monitor sales performance over time by region, state, and product category.  
- **Profitability Metrics:** Identify products, customers, and regions generating the most profit.  
- **Customer Insights:** Analyze customer behavior and purchase patterns.  
- **Interactive Filters:** Filter by region, state, product category, or order channel.
